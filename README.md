## Installation

```
$ git clone git@github.com:BoisselNicolas/GestionStock.git
```

## Compilation


```
$ cd GestionStock
$ dotnet build
```

## Usage 

Pour éxécuter le programme

```
$ dotnet run 
```

## Test unitaire 

Pour lancer les tests unitaire

```
$ cd GestionStockTest
$ dotnet test 
```
